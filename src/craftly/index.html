FAQ

Honest answers to the questions you're already asking.

Ehrliche Antworten auf Ihre Fragen.

Can I really download and host the code myself?Kann ich den Code wirklich selbst hosten?

Yes. Every project is a real Laravel or WordPress codebase you can git clone and composer install. Deploy to Hetzner, Forge, your own VPS — wherever PHP runs.

Ja. Jedes Projekt ist eine echte Codebase, die Sie klonen und mit composer install starten können. Hetzner, Forge, eigener VPS — überall, wo PHP läuft.

What happens after the beta?Was passiert nach der Beta?

Beta seats keep their early-access pricing forever (50% off Pro). After GA, plans start at €19/month with credit-based generation top-ups available.

Beta-Plätze behalten den Early-Access-Preis dauerhaft (50% Rabatt auf Pro). Nach GA starten Pläne ab 19 €/Monat.

Why PHP and not React?Warum PHP und nicht React?

PHP runs ~75% of the web. Every shared host speaks it. Your client's agency speaks it. Your CMS speaks it. We meet you where production already lives.

PHP läuft auf ~75% des Webs. Jedes Shared Hosting spricht es. Wir treffen Sie dort, wo die Produktion schon läuft.

Is the code actually good, or just plausible-looking?Ist der Code wirklich gut?

The Validation Fix agent runs PHPStan, Larastan, and our own conventions before each preview. Generated code passes php artisan test, follows PSR-12, uses Eloquent properly.

Der Validation-Fix-Agent prüft mit PHPStan, Larastan und eigenen Konventionen vor jeder Preview. Code besteht php artisan test, folgt PSR-12.

GDPR & data residency?DSGVO & Data Residency?

Built in Germany. Data processed in EU regions. AVV (DPA) on request. We don't train on your prompts or code.

Gebaut in Deutschland. Datenverarbeitung in EU-Regionen. AVV auf Anfrage. Kein Training auf Ihren Daten.

What about more complex apps — multi-tenant, payment, integrations?Komplexere Apps — Multi-Tenant, Payment, Integrationen?

Stripe, OAuth, multi-tenant scaffolds, queue workers — supported. Beyond that, the codebase is yours; bring it to your team or back to ShvedkoDev for a follow-on engagement.

Stripe, OAuth, Multi-Tenant-Scaffolds, Queue-Worker — unterstützt. Darüber hinaus gehört der Code Ihnen.
